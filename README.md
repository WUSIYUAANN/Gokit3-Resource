## 二、概要

   `Gokit3-Resource`是机智云官方的`GoKit3 soc`开发板的资料汇总，里面包含机智云提供开发板的各种驱动文件，以及如何接入机智云的API文档。`GoKit3`开发板分为3种，当前的是`soc`版，即为`esp8266`为通讯模组。
